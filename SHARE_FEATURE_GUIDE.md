# Share Feature User Guide

## Overview

The share functionality allows you to share your photo albums with clients in two ways:

1. **Public Link** - Anyone with the link can access the album
2. **Private Share** - Only specific people you invite can access the album

### View Active Shares

## Client Experience

### Accessing Public Links

1. Client receives your share link
2. Opens link in browser
3. If password-protected:
   - Enters password
   - Gets access to album
4. Views photos with allowed permissions

### Accessing Private Links

1. Client receives unique private link
2. Opens link in browser (no password needed)
3. Views album with configured permissions
4. Each access is tracked individually

## Troubleshooting

### "Album not found" error

**Causes:**
- Link has expired
- Share was revoked
- Album was deleted

**Solution:** Contact the photographer for a new link

### "Invalid password" error

**Cause:** Incorrect password entered

**Solution:** 
1. Double-check password (case-sensitive)
2. Contact photographer if forgotten

### Photos not loading

**Causes:**
- Slow internet connection
- Large photo files
- Browser issues

**Solution:**
1. Refresh the page
2. Try a different browser
3. Check internet connection

### Can't download photos

**Cause:** Download permission not enabled

**Solution:** Photographer needs to update share permissions
